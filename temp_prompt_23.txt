Glossar

0–9

0-Euro-Cockpit: Eine selbstgebaute, kostenlose digitale Kommandozentrale für das eigene Unternehmen, die ohne teure Lizenzen auskommt und lediglich einen Webbrowser und Texteditor benötigt.

A

API (Application Programming Interface): Eine Programmierschnittstelle, die wie ein digitaler "Kellner" fungiert. Sie vermittelt Daten und Anfragen zwischen dem Dashboard und externen Diensten (wie einer KI oder Datenbank), ohne dass die Webseite neu geladen werden muss.

B

BPMN (Business Process Model and Notation): Ein weltweiter Industriestandard zur formalen und rechtssicheren Modellierung von Geschäftsprozessen. Arbeitet visuell mit sogenannten Swimlanes (Zuständigkeiten) und Gateways (Abzweigungen).

C

C4-Modell: Ein pragmatisches IT-Architektur-Modell (Context, Container, Components, Code), das IT-Landschaften und Abhängigkeiten visualisiert wie eine zoombare Landkarte – vom groben System-Kontext bis tief in den Code.

CDN (Content Delivery Network): Ein globales Netzwerk zur Auslieferung von Inhalten. Wird genutzt, um externe Code-Bibliotheken (z. B. für Diagramme) direkt in die HTML-Datei deines Cockpits einzubinden, ohne sie vorher auf die eigene Festplatte herunterladen zu müssen.

Chart.js: Eine beliebte und einsteigerfreundliche JavaScript-Bibliothek, um Zahlen in HTML-Dateien als ansprechende Torten- oder Balkendiagramme zu visualisieren.

Cockpit (vs. Dashboard): Während ein reines Dashboard Daten lediglich anzeigt, ist ein Cockpit darauf ausgelegt, Prozesse aktiv zu steuern (Kommandozentrale). Das Dashboard ist somit ein funktionaler Baustein des Cockpits.
